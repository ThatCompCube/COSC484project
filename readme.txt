How to Run
-----------
* Download the current repository.
* Open a terminal to the contents of the repository.
* Type in "npm i".
* Afterwards, type in "node server.js".
* Once the server is running, you can enter the website through mainPage.html in your browser.
* You will need to register an account to enter the mainPage. The register button is found in the login screen.

-----------------------
Contents of the Project
-----------------------

* Functioning Login screen and Registration screen, including log out feature on the main page
* Main page listing links to Schools and Teacher profiles
